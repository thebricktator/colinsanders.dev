# colinsanders.dev

My personal portfolio site. Built with Next.js, Tailwind CSS, and vibes.

## Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Font**: Space Grotesk
- **Hosting**: Vercel

## Development

```bash
npm install
npm run dev
```
